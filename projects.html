Rocket Engine Analysis

Over the past six months I've developed a Python system that can design an rocket engine contour and regenerative cooling system based on desired thrust, conraction ratio and chamber pressure. I am actively using this codebase to create preliminary designs for a ~3000lb liquid rocket engine that aims to take a student built rocket to space.
